INFORMATION

Welcome to shootgoose studios! My name is Elijah Mylius-Aceves, and I am the developer who runs the organization. An extensive list of my personal and collaborative projects can be found here, in addition to contact information of all kinds. Please enjoy your stay!

PROJECTS

Townhold

Townhold is a 2D sandbox city-builder themed around creating a town and watching it grow throughout historical ages. The game combines a relaxing atmosphere with cute art and beautiful music made by OhEight with the main gameplay loop of games like Civilization VI and Cities:Skylines.

The game is officially released but will still get periodic updates on itch; the link can be found here:

Townhold on Itch.io

Goblin Ring

Goblin Ring is a local multiplayer versus rougelike pitting two places in very close proximity to each other. Centered around goblins, players must fight each other to become the next KING GOBLIN.

Come back later for a link; the game is still in active development!

Black Tower [In Active Development]

Black Tower is my current collaborative game project with the Black Tower Team. While still in active development, the game is a rougelike dungeon-crawler (I know, there's not enough of those) but with a particular emphasis on multiplayer!

This game is being worked on as a semester project for the SJSU Game Development Club. The link to an early version can be found here:

Black Tower on Itch.io

Last Chance

Last Chance is a game made for the 2025 Global Game Jam made with four other very talented developers. It centeres around vertical movement on a 2D screen. You play as a bubble trying to escape the abstractions of a post-human world!

The game can be found here:

Last Chance on Itch.io

Iris

Iris is my first collaborative game project and is a puzzle/adventure game centering around a girl named Iris who is mysteriously awakened in a Panopticon-like prison. The game is short but acts as a technical demo for future 2D collaborative works.

Iris on Itch.io
